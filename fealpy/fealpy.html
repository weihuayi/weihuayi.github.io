FEALPy 帮助与安装

学习与帮助

入门基础

使用 FEALPy 你需要有一些关于计算机的基础知识和技能, 比如操作系统, 命令行终端, 安 装和删除软件等.

当然你也要有基础的数学知识, 如数学分析, 高等代数, 解析几何, 计算数学等方面的知识.

另外, 编程的第一手资料基本都是英文的, 程序执行 报错也是英文的, 所以还要你愿意看, 而且看得懂英语的文献.

有了这些知识, 你再把下面两本书的看一看, 最起码要把基础的东西看一看.

  • Git 教程(中文)
  • Python 科学计算讲义(英文)

有了上面的基础, 你就可以安装使用 FEALPy 了. 后面需要你在不断的在实践中学习提高了 . 每天尝试写点程序, 来解决自己学业和研究中的问题. 我始终坚信, 只有融入了自己亲身 经验的知识才是真正的个人能力.

纸上得来终觉浅，绝知此事要躬行。--陆游

安装

Ubuntu

  1. 安装 Python3 解释器: 
    $ sudo apt install git            # 版本控制 
$ sudo apt install python3        # Python3 解释器 
$ sudo apt install python3-pip    # Python3 软件包管理器
    利用 pip3 安装 Python3 的科学计算环境 (尽量不用 ubuntu 的 apt 安装, apt 装的包 可能不是最新的，存在不兼容的问题)
    $ sudo -H pip3 install ipython # 高级交互式解释器 
$ sudo -H pip3 install numpy  # 多维数组及运算 
$ sudo -H pip3 install scipy  # 高级科学计算软件包 
$ sudo -H pip3 install matplotlib # 2D 和 3D 画图 
$ sudo -H pip3 install pybind11 # python 与 c++ 接口
$ sudo -H pip3 install pyamg # 多重网格解法器
$ sudo -H pip3 install meshpy # 网格生成器
$ sudp -H pip3 install meshio # 常见的网格文件格式读取
  2. 配置你的 git 信息:
    $ git config --global user.name "Your Name"
$ git config --global user.email "Your Email"
  3. 从 github 或 gitlab 上 clone 最新的 FEALPy:
    $ git clone https://github.com/weihuayi/fealpy.git # or
$ git clone https://gitlab.com/weihuayi/fealpy.git
  4. 进入 fealpy/, 运行下面命令安装: 
    $ sudo python3 setup_linux.py develop
    这样会把 FEALPy 以符号链接的方式安装到目录 /usr/local/lib/python3.<x>/site-packages/，注意其中 <x> 在不同的系统下可能不 同。 这样安装的好处是, 每次在 fealpy/ 目录下运行
    $ git pull # 更新 FEALPy 的到最新版本
    你就可以在其它的任何目录下调用 FEALPy 了.

Windows

  1. 下载最新的 Anaconda for Windows.
  2. 下载最新的 Git for Windows.
  3. 开始菜单中在 Anaconda 菜单目录下打开 Anaconda Prompt (Python) 命令行终端, 注 意它会默认进入你的 Desktop 的上一级目录. 然后配置你自己的 Git 信息.
    git config --global user.name "Your Name"
git config --global user.email "Your Email"
  4. 进入 Desktop 目录, 从 gitlab 或 github 克隆最新的 FEALPy 
    cd Desktop # 进入主目录
git clone https://github.com/weihuayi/fealpy.git # or 
git clone https://gitlab.com/weihuayi/fealpy.git
  5. 进入 FEALPy, 运行下面的命令安装:
    cd fealpy
python setup_win.py develop 
pip install https://download.lfd.uci.edu/pythonlibs/s2jqpv5t/MeshPy-2018.2.1-cp37-cp37m-win_amd64.whl
pip install https://download.lfd.uci.edu/pythonlibs/s2jqpv5t/pyamg-4.0.0-cp37-cp37m-win_amd64.whl
    注意这里是安装的是 Python 3.7 版本. 如果你安装的 Python 是其它版本，请相应更换掉 .
  6. 最后, 你在开始目录中可以打开 Python 集成开发环境 Spyder, 就可以基于 FEALPy 写 你的代码了.

Mac OS

  1. 下载安装最新的 Anaconda for macOS.
  2. 下载安装最新的 Git for Mac.
  3. 打开终端配置你的 Git 信息. 
    $ git config --global user.name "Your Name"
$ git config --global user.email "Your Email"
  4. 从 github 或 gitlab 上 clone 最新的 FEALPy
    $ git clone https://github.com/weihuayi/fealpy.git 
$ git clone https://gitlab.com/weihuayi/fealpy.git
  5. 进入 fealpy/, 运行下面的命令安装 FEALPy:
    $ sudo python setup_mac.py develop

pip 管理你需要的软件包

按照上面的安装过程中, 可能会会显示一些依赖的包的没有安装(一般用红色字体的英文显 示), 欢迎反馈信息给我. 我挣 取把安装脚本写的更智能一点. 但你也可以自己动手, 用 Python 自己的软件包管理命令 pip (Ubuntu 上是 pip3, Anaconda (Windows) 中的命令是 pip) 解决问题.

在 Ubuntu 下, 比如说没有 boost 这个包, 用下面的命令就可以安装:

$ pip3 search boost
$ sudo -H pip3 install boost

在 Anaconda (Windows) 下,

$ pip search boost
$ pip install boost
