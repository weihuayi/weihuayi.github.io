Zoom 的使用中的问题与解决方案

Zoom 是目前我知道的唯一一个可以在 Ubutnu 下开视频会议的软件.

Ubuntu 下的安装

https://us04web.zoom.us/

run zoom successfully,exit code is 0

$ killall zoom

错误代码 200

中国区域注册的帐号 5 月 1 号以后无法开启新的会议

利用 google 帐户注册一个 zoom.us 的账号即可.
