个人简介

魏华祎，男，1984年10月4号出生于河南省邓州市， 2002 年进入河南大学数学与信息科学 学院应用数学专业学习，2006 毕业后进入湘潭大学计算数专业硕博连续，并于 2011 年前 往美国加州大学欧文分校博士联合培养学习一年，于 2012 年毕业于湘潭大学数学与计算科 学学院，获理学博士学位，并留校工作，现为湘潭大学数学与计算科学学院副教授。

现主要从事科学计算方面的研究工作，精通 Linux 操作系统及相关软件工具，同时精通 C++、 Python 和 Matlab 三门编程语言，专注于开发高效数值计算开源软件， 致力于把所 学数学知识转化为实际生产力，服务于实际工业应用。

我每年会招收 2-3 名左右的硕士研究生，下一年应该也可以招收博士生了。欢迎真正想学 习并提高自我能力的学生报考我。在带研究生的过程中，我会同时注重计算数学理论和动手 实践能力的培养，特别是编程能力的培养。学习优秀者，硕士生未来可以继续读博深造，或 者进入高科技企业工作。

如果想跟着我读研究生，必须首先完成如下的面试题目：

在数据科学学习网站 www.dataquest.io 上注册一个帐户，选择一个学习路径，学习 Python 编程基础。

网站上的学习资料是英语的，提供编程环境（不需要你自己在本地搭建）。知识点由易到难 ，只有完成当前知识点，并完成编程练习题目才能学习下面新的知识点。每个新知识点都比 较小，认真做并不需要太多的时间即可完成。

要求每天邮件向我汇报学习进程，包括学到知识总结，疑问及完成练习的截图。我不要求参 与者每天学很多知识点，一个就足够了。

上面的面试题目主要考察：

  1. 英语阅读能力
  2. 自主学习能力
  3. 动手实践能力
  4. 学习态度，是否积极，是否能够坚持把一件事情做好

如果你想来，就现在开始做这个题目吧。
